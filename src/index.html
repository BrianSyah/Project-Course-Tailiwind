About Me

Halo, saya Brian, seorang Fullstack Developer Web yang memiliki pengalaman dalam membangun aplikasi web yang dinamis dan skalabel. Saya menguasai teknologi frontend dan backend untuk menciptakan pengalaman pengguna yang intuitif sekaligus memastikan sistem yang kuat di belakang layar.

Keahlian saya meliputi teknologi frontend: seperti React, Next.js, atau Vue.js untuk membuat antarmuka yang menarik, serta teknologi backend: seperti Node.js, Express, atau Laravel untuk mengembangkan fungsi server yang andal.
